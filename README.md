Megamania Remake
Jogo de tiro espacial em pixel art inspirado no clássico Megamania, com ondas misturadas de inimigos, munição especial, energia constante e suporte para desktop e mobile.
Como rodar
Abra index.html em um navegador moderno.
Se preferir, publique os arquivos no GitHub Pages apontando para o branch main na raiz do repositório.
Controles
Desktop:← e → movem a nave
Espaço atira
P pausa
Enter pausa

Mobile:botões na tela para mover, atirar e pausar

Jogabilidade
A barra de energia cai continuamente
Se a energia zerar, o jogador perde uma vida
Destruir toda a onda recupera a energia e avança de fase
A velocidade dos inimigos aumenta a cada nível
As ondas misturam sprites diferentes na mesma fase
Inimigos podem soltar bônus de munição no meio da partida
Armas especiais
Laser: tiro padrão
Spread: dispara três projéteis em leque
Piercing: projétil atravessa múltiplos inimigos
Rapid: reduz o intervalo entre disparos
Interface
Score no topo
Barra de energia na parte inferior
Ajuda de controles na tela
Pausa com sobreposição e retorno sem perder o estado
Observação
O jogo foi pensado para funcionar bem em tela fixa, com visual arcade e colisão precisa no estilo retrô.
